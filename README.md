## 📥 Cara Instalasi

Ikuti langkah mudah ini untuk menjalankan proyek di komputer lokal Anda:

1. **Klon repositori ini**
   ```cmd
   git clone https://github.com
   ```
2. **Masuk ke direktori proyek**
   ```cmd
   cd nama-proyek
   ```
3. **Instal dependensi**
   ```cmd
   npm install
   ```
4. **Jalankan website**
   ```cmd
   npm run dev
   ```
