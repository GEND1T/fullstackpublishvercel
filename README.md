## 📥 Cara Instalasi

Ikuti langkah mudah ini untuk menjalankan proyek di komputer lokal Anda:

1. **Klon repositori ini**
   ```cmd
   git clone https://github.com/GEND1T/fullstackpublishvercel.git
   ```
2. **Masuk ke direktori proyek**
   ```cmd
   cd nama-proyek
   ```
3. **Masuk ke direktori frontend**
   ```cmd
   cd frontend
   ```
4. **Instal dependensi**
   ```cmd
   npm install
   ```
5. **Jalankan website**
   ```cmd
   npm run dev
   ```
6. **Lakukan Ulang untuk menjalankan backend**
